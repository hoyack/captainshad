# BayWise Fishing Dataset — Plan

## Objective
Create a JSON dataset of fish species and their seasonal availability (by month) for the Aransas Bay system (Aransas Pass, Port Aransas, Rockport, Texas).

## Stage 1 — Research
Deploy parallel research agents to gather seasonal fishing data for the Aransas Bay system:
- Agent 1: Research inshore bay species (redfish, speckled trout, black drum, flounder, etc.)
- Agent 2: Research nearshore/offshore species (snapper, kingfish, tarpon, sharks, etc.)
- Agent 3: Research seasonal patterns, peak months, and migration timing

## Stage 2 — Synthesis & JSON Construction
Compile all research into a structured JSON dataset with species, peak months, and notes.

## Output
- JSON file: `/mnt/agents/output/baywise_fishing_dataset.json`
